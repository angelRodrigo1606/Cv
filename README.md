# Portafolio Terminal Style - Hacker Theme

Un CV interactivo con estética de terminal/hacking, efectos Matrix y diseño futurista en verde neón.

# El CV incluye las siguientes secciones personalizables:

PERFIL_PROFESIONAL - Resumen de tu perfil
EXPERIENCIA_LABORAL - Historial de trabajos
HABILIDADES_TÉCNICAS - Skills divididos por categorías
PROYECTOS_DESTACADOS - Portfolio de proyectos
EDUCACIÓN - Formación académica y certificaciones

## Responsive Design
El CV es completamente responsive y se adapta a:

- Móviles (< 768px)
- Tablets (768px - 1024px)
- Desktop (> 1024px)
